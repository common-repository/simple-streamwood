Enables StreamWood widget on all pages.

== Description ==
This plugin adds the required javascript for StreamWood widget.

For more information visit: [StreamWood](https://streamwood.ru/)

== Installation ==
1. Upload `simple-streamwood` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add the StreamWood Key and StreamWood Domain Key from StreamWood site to the settings (Admin > Settings > Simple StreamWood).
